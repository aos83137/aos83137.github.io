
# Project Overview

This project is a static wedding invitation website for Yongseok and Yumeno. It is built with HTML, CSS, and JavaScript. The website is available in two languages: Korean and Japanese.

## Key Features:

*   **Dual Language Support:** The website has separate pages for Korean (`ko/index.html`) and Japanese (`jp/index.html`) guests.
*   **Interactive Elements:** The website includes a countdown timer, an image gallery with a lightbox, and background music with a play/pause control.
*   **RSVP:** Guests can RSVP via a Google Form linked on the website.
*   **Notion Integration:** The `package.json` file includes the `@notionhq/client`, which suggests a potential integration with Notion, although the exact usage is not clear from the existing files.

# Building and Running

This is a static website, so there is no build process. To run the project, simply open the `ko/index.html` or `jp/index.html` file in a web browser.

# Development Conventions

The project uses vanilla JavaScript for its interactive elements. The code is well-structured and easy to understand. The project also uses a variety of fonts from Google Fonts.
